# 💻 Bank Management System (C++)

## 📌 Description
This is a simple Bank Management System developed using C++ and Object-Oriented Programming concepts. It allows users to manage bank accounts with features like creating accounts, depositing, withdrawing, and searching accounts.

---

## 🚀 Features
- Create Account
- Multiple Accounts Support
- Deposit Money
- Withdraw Money
- Search Account
- Display All Accounts
- File Handling (Data stored permanently)

---

## 🧠 Concepts Used
- Object-Oriented Programming (OOP)
- Class and Object
- Encapsulation
- File Handling (Binary Files)

---

## 🛠️ Technologies Used
- C++
- File Handling

---

## ▶️ How to Run
1. Open project in VS Code
2. Compile:
   g++ main.cpp -o main
3. Run:
   main
